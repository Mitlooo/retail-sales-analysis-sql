# Retail SQL Analysis Project

This is a beginner SQL project using MySQL. It uses the same type of sales data as the Excel dashboard project, but the analysis is done using SQL queries instead of PivotTables.

## Goal

Learn how to use SQL to answer business questions about sales, profit, products, regions, cities, delivery status, and payment methods.

## Tools

- MySQL
- MySQL Workbench

## Files

- `01_create_database_and_table.sql`: creates the database, table, and sample data
- `02_practice_questions.sql`: questions for you to try
- `03_answers.sql`: answers to check after you try
- `sql_learning_notes.md`: simple explanations of SQL concepts
- `project_explanation.md`: project summary for CV, GitHub, and interviews

## How To Start

1. Open MySQL Workbench.
2. Open `01_create_database_and_table.sql`.
3. Run the full script.
4. Confirm that the table shows 40 rows.
5. Open `02_practice_questions.sql`.
6. Try each query slowly.
7. Use `03_answers.sql` only to check your work.
